# คู่มือเริ่มต้นใช้งานโปรแกรมออกแบบโครงสร้างเหล็ก

## การติดตั้งและใช้งาน

### 1. ติดตั้งโปรแกรม

```bash
# ติดตั้ง dependencies ที่จำเป็น
pip install -r requirements.txt

# รันโปรแกรม
python main_app.py
```

### 2. หน้าจอหลักของโปรแกรม

โปรแกรมมี 5 แท็บหลัก:

| แท็บ | ใช้งานสำหรับ |
|------|-------------|
| 🏠 **ออกแบบแป (Purlin)** | ออกแบบแปหลังคาเหล็กรูปตัวซี |
| 📏 **ออกแบบคาน (Beam)** | ออกแบบคานเหล็กเอชบีม/ไอบีม |
| 🏛️ **ออกแบบเสา (Column)** | ออกแบบเสารับแรงอัดและโมเมนต์ |
| 🔩 **รอยต่อ (Connection)** | ออกแบบรอยต่อสลักเกลียวและรอยเชื่อม |
| 🔲 **แผ่นฐาน (Base Plate)** | ออกแบบแผ่นฐานเสา |

---

## วิธีใช้งานแต่ละแท็บ

### 🏠 แท็บที่ 1: ออกแบบแป (Purlin)

**ขั้นตอน:**
1. เลือกหน้าตัดเหล็ก (เช่น C150x65x20x4.0)
2. ป้อนความยาวช่วงแป (เมตร)
3. ป้อนระยะห่างระหว่างแป (เมตร)
4. ป้อนมุมลาดชันของหลังคา (องศา)
5. ป้อนน้ำหนักบรรทุก:
   - น้ำหนักบรรทุกคงที่ (kPa)
   - น้ำหนักบรรทุกจร (kPa)
6. ป้อนข้อมูลลม:
   - ความเร็วลม (m/s) - กรุงเทพฯ 30 m/s
   - ความสูงอาคาร (m)
   - ประเภทสภาพภูมิ (B, C, D)
7. กดปุ่ม **"คำนวณ"**

**ผลลัพธ์:**
- ✅ ผ่าน = หน้าตัดสามารถรับน้ำหนักได้
- ❌ ไม่ผ่าน = หน้าตัดไม่สามารถรับน้ำหนักได้

**ปุ่ม "ค้นหาหน้าตัดประหยัดสุด":**
- โปรแกรมจะหาหน้าตัดที่เล็กที่สุดที่ยังผ่านเกณฑ์การออกแบบ

---

### 📏 แท็บที่ 2: ออกแบบคาน (Beam)

**ขั้นตอน:**
1. เลือกหน้าตัดเหล็ก (H-Beam หรือ I-Beam)
2. ป้อนช่วงคาน (เมตร)
3. เลือกประเภทคาน:
   - Simply Supported = คานรองรับ 2 ปลาย
   - Cantilever = คานยื่น
4. เลือกการยึดหน่วงข้าง:
   - continuous = ยึดหน่วงตลอดช่วง
   - ends_only = ยึดหน่วงเฉพาะปลาย
5. ป้อนน้ำหนัก:
   - Dead Load (kN/m) = น้ำหนักคงที่
   - Live Load (kN/m) = น้ำหนักจร
   - Point Loads (kN) = น้ำหนักจุดที่กลางคาน
6. กดปุ่ม **"คำนวณคาน"**

**การตรวจสอบ:**
- ✅ หน่วยแรงดัด (Bending Stress)
- ✅ หน่วยแรงเฉือน (Shear Stress)
- ✅ การแอ่นตัว (Deflection)

---

### 🏛️ แท็บที่ 3: ออกแบบเสา (Column)

**ขั้นตอน:**
1. เลือกหน้าตัดเหล็ก (H-Beam, I-Beam, หรือท่อ)
2. ป้อนความสูงเสา (เมตร)
3. ป้อนค่า K (Effective Length Factor):
   - Kx = 1.0 (ปกติ)
   - Ky = 1.0 (ปกติ)
4. ป้อนน้ำหนัก:
   - Axial Loads (kN) = แรงอัดตามแนวแกน
   - Moments (kN-m) = โมเมนต์รอบแกน X และ Y
5. กดปุ่ม **"คำนวณเสา"**

**การตรวจสอบ:**
- ✅ อัตราส่วนความชะลูด (KL/r ≤ 200)
- ✅ แรงอัดแกน (Axial Compression)
- ✅ แรงรวม (Combined Loading + Interaction)

---

### 🔩 แท็บที่ 4: รอยต่อ (Connection)

#### ส่วนที่ 1: สลักเกลียว (Bolted)

**ขั้นตอน:**
1. เลือกขนาดและเกรดสลักเกลียว (เช่น M20 A325)
2. ป้อนจำนวน bolts
3. เลือกระนาบเฉือน:
   - Single = ระนาบเดียว
   - Double = 2 ระนาบ
4. ป้อนความหนาแผ่นต่อ (mm)
5. ป้อนระยะถึงขอบ (mm)
6. ป้อนแรงเฉือน (kN)
7. กดปุ่ม **"คำนวณรอยต่อสลักเกลียว"**

#### ส่วนที่ 2: รอยเชื่อม (Welded)

**ขั้นตอน:**
1. เลือกประเภทลวดเชื่อม (E60XX, E70XX, E80XX)
2. ป้อนขนาดรอยเชื่อม (mm)
3. ป้อนความยาวรอยเชื่อม (mm)
4. ป้อนแรงเฉือน (kN)
5. กดปุ่ม **"คำนวณรอยเชื่อม"**

---

### 🔲 แท็บที่ 5: แผ่นฐาน (Base Plate)

**ขั้นตอน:**
1. เลือกหน้าตัดเสา
2. ป้อนขนาดแผ่นฐาน:
   - ความกว้าง B (mm)
   - ความยาว N (mm)
   - ความหนา tp (mm)
3. ป้อนกำลังคอนกรีต f'c (MPa) - ปกติ 24 MPa
4. ป้อนน้ำหนัก:
   - Axial Loads (kN)
   - Moments (kN-m)
5. กดปุ่ม **"คำนวณแผ่นฐาน"**

**การตรวจสอบ:**
- ✅ แรงตปะคอนกรีต (Concrete Bearing)
- ✅ ความหนาแผ่นฐาน (Plate Thickness)

## เกณฑ์การออกแบบตามมาตรฐาน วสท.

### อัตราส่วนการตรวจสอบ

| รายการ | เกณฑ์ | หมายความว่า |
|--------|-------|-------------|
| Stress Ratio | ≤ 1.0 | หน่วยแรงที่เกิดขึ้น ≤ หน่วยแรงที่ยอมให้ |
| Deflection Ratio | ≤ 1.0 | การแอ่นตัว ≤ การแอ่นตัวที่ยอมให้ |
| Interaction Ratio | ≤ 1.0 | แรงรวม (อัด+ดัด) ≤ ความสามารถรับน้ำหนักรวม |

### ค่าจำกัดการแอ่นตัว

| สมาชิก | เงื่อนไข | จำกัด |
|--------|----------|-------|
| คาน | Live Load | L/360 |
| คาน | Total Load | L/240 |
| แป | Live Load | L/240 |
| แป | Total Load | L/180 |
| เสา | ยอดเสา (ลม/แผ่นดินไหว) | H/300 |
